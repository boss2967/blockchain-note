7.3 jquery属性操作

  • 1. jquery属性操作
    • 1.1. html() 取出或设置html内容
    • 1.2. prop() 取出或设置某个属性的值
    • 1.3. 练习

1. jquery属性操作

1.1. html() 取出或设置html内容

// 取出html内容

var $htm = $('#div1').html();

// 设置html内容

$('#div1').html('<span>添加文字</span>');

1.2. prop() 取出或设置某个属性的值

// 取出图片的地址

var $src = $('#img1').prop('src');

// 设置图片的地址和alt属性

$('#img1').prop({src: "test.jpg", alt: "Test Image" });
