1. 事件冒泡

1.1. 什么是事件

1.2. 事件冒泡的作用

1.3. 阻止事件冒泡

事件冒泡有时候是不需要的，需要阻止掉，通过event.stopPropagation();来阻止

1.4. 阻止默认行为

阻止表单提交

$('#form1').submit(function(event){
    event.preventDefault();
})

1.5. 合并阻止操作

event.stopPropagation();
event.preventDefault();
return false;

注意有区别：

return false;

后面的语句不会执行

event.stopPropagation();
event.preventDefault();

后面的语句会执行
