# 命名规范
1. 区分大小写
2. 第一个字符必须是字母，下划线(_)或者美元($)
3. 其他字符可以是字母，下划线，美元

# 匈牙利命名风格：

* 对象o Object 比如：oDiv

*  数组a Array 比如：aItems

* 字符串s string 比如：sUserName
