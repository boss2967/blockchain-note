Js是一种弱类型语言，js的变量类型由它的值来决定。

js,html,css会下载到用户电脑上，用浏览器去运行，和后端代码在服务器上运行不一样。 js代码会把代码合并成一行，需要加；去短句。

1. 数据类型

//定义变量
/*

var iNum01 = 12,sTr = "abc",bIsMove =true;

1.1. 五种基本数据类型：

  1. number
  2. string
  3. boolean
  4. undefined
  5. null

1.2. 一种复合类型

object（数组，字段都属于object类型）

2. 命名规范

  1. 区分大小写
  2. 第一个字符必须是字母，下划线(_)或者美元($)
  3. 其他字符可以是字母，下划线，美元

3. 匈牙利命名风格：

  • 对象o Object 比如：oDiv

  • 数组a Array 比如：aItems

  • 字符串s string 比如：sUserName
