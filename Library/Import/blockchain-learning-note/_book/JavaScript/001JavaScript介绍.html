1. JavaScript介绍

JavaScript 是运行在浏览器的脚本语言，js主要解决的是前端与用户交互的问题，包括使用交互与数据交互，JavaScript是浏览器解释执行的。

前端三大块:

  1. HTML：页面结构
  2. CSS：页面表现：元素大小，颜色，位置，隐藏或显示，部分动画效果
  3. JavaScript：页面行为：部分动画效果，页面与用户交互，页面功能
